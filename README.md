по идее это будет анализатор отзывов

- на данный момент реализован фронтенд и часть ьбекенда на django
-- сейчас из веб чассти можно написать отзыв и он уходит в бекенд и созраняется в БД  PostgreSQL.

примеры запросов:

GET http://127.0.0.1:8000/api/feedback/   
в ответ мы получим пустой список

POST http://127.0.0.1:8000/api/feedback/  
вкладываем в него  JSON { "text": "Отличный товар!" }
и это сохранится
